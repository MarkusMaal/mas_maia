Seadme tuvastamine nurjus

Seadet ei õnnestunud automaatselt tuvastada. Kui see juhtus seadmega, mis on juba valges nimekirjas, proovige uuesti. Kui see seade pole sertifitseeritud Markuse asjade seade, lisage see lubatud seadmete hulka käsitsi. Selleks tehke järgnevat:

  • Lukustage Markuse arvuti ekraan lahti
  • Paremklikkige tegumiriba olekualal
  • Avage alammenüü "M.A.I.A. serveri haldamine"
  • Klõpsake avanenud menüüs "Ava brauseris"
  • Avanenud veebilehel klõpsake "Lisa seade"
  • Sisestage seadme IP aadress, mille leiate selle lehekülje allosast
  • Valige seadme tüüp
  • Klõpsake "Lisa seade"
  • Vajutage siin seadmes nuppu "Proovi uuesti"

Tuvastamata seadmed saavad näha ainult teavet arvuti kohta ning Markuse asjade versioone.
